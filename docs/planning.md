# 노고 아카데미 포크레인 RC 조종 가이드 — 기획 문서

## 배경

속초 해수욕장에 위치한 노고 아카데미(포크레인 RC 카페)에서 손님들이 조종 방법을 어려워하는 문제가 있었다.
직원이 매번 직접 설명해야 했고, 잘못된 조작으로 인한 고장도 종종 발생했다.

QR 코드를 통해 손님이 스스로 조종 방법을 확인할 수 있는 가이드 페이지를 만들기로 했다.

---

## 목적

- 직원의 반복 설명 부담 줄이기
- 손님이 헷갈릴 때 폰으로 스스로 확인 가능하게
- 잘못된 조작으로 인한 고장 감소
- 체험 만족도 향상

---

## 설계 방향

### 기술 스택

| 항목 | 결정 | 이유 |
|------|------|------|
| 구현 방식 | 단순 HTML/CSS | 서버 불필요, 유지보수 쉬움 |
| 호스팅 | GitHub Pages | 완전 무료, 별도 서버 없음 |
| URL | seohaun.github.io/noggo-guide | QR 코드로 연결 |

### 페이지 구성

1. **주의사항** — 체험 전 필독 (고장 유발 행동 안내)
2. **왼쪽 조이스틱** — 이동 조작
3. **오른쪽 조이스틱** — 상체 회전 / 붐 암 조작
4. **버튼** — 버킷(바가지) 열고 닫기
5. **잘못된 조작 예시** — 고장 원인 영상

### 영상 구성

- 각 조작마다 짧은 영상 클립 삽입
- 여자친구가 직접 촬영 예정
- 영상 준비 전까지 플레이스홀더로 대체

### 디자인 원칙

- **모바일 최적화** — 손님이 폰으로 QR 찍어서 봄
- 어두운 헤더 + 카드 레이아웃
- 초록색 포인트 (포크레인 색상 통일)
- 텍스트 최소화, 영상 중심

---

## 남은 작업

### 여자친구가 할 것

- [ ] 각 조작별 영상 촬영 (왼쪽 조이스틱, 오른쪽 조이스틱, 버킷, 잘못된 조작)
- [ ] GitHub Pages 활성화 (Settings → Pages → main 브랜치)

### 개발 작업

- [ ] 영상 파일 업로드 후 video 태그 연결
- [ ] 실제 조종기 버튼 설명 텍스트 업데이트 (HUINA 모델 기준)
- [ ] QR 코드 생성 및 카페 내 부착
- [ ] (선택) 영어 버전 추가 — 외국인 관광객 대응

---

## 향후 고려사항

- 예약 시스템: 지금은 불필요, 손님이 늘면 네이버 예약 연동 검토
- 다국어: 속초 특성상 외국인 방문객 있을 경우 영어 페이지 추가
