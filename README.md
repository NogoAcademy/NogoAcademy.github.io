## 📄 페이지 구성 순서

1. **헤더** — 브랜드명, 페이지 제목, 한 줄 안내
2. **매장 사진** — 체험 공간 소개
3. **핵심 주의사항** — 플레이 전 체크 (4가지 공식 안내)
4. **멈췄을 때 대처법** — 직원 호출 안내
5. **처음이라면 이것부터** — 5단계 체험 순서
6. **조작 방법** — 왼쪽/오른쪽 조이스틱 카드 + 영상
7. **조종기 가이드 사진** — 전체 컨트롤 레이아웃 참고
8. **하지 말아야 할 조작** — 경고 항목
9. **이용 제한 안내** — 손해배상 고지
10. **푸터** — 직원 호출 안내

## 🎮 실제 조종기 조작법

### 왼쪽 조이스틱 (파란색)
| 방향 | 동작 |
|------|------|
| ↑ | 암 뻗기 (Arm Out) |
| ↓ | 암 당기기 (Arm In) |
| ← → | 스윙 — 상체 좌우 회전 (Swing) |

### 오른쪽 조이스틱 (주황색)
| 방향 | 동작 |
|------|------|
| ↑ | 붐 내리기 (Boom Down) |
| ↓ | 붐 올리기 (Boom Up) |
| ← | 버킷 닫기 (Bucket Curl) |
| → | 버킷 열기 (Bucket Dump) |

> ⚠️ 오른쪽 스틱은 ↑가 붐 내리기라 반직관적으로 느껴질 수 있음

## 🎬 영상 교체 방법

각 조작법 카드에 영상을 넣으려면 `index.html` 에서 아래 부분을 찾아 수정해요.

**현재 (영상 없음):**
```html
<div class="video-wrap no-video"></div>
```

**영상 추가 시:**
```html
<div class="video-wrap has-video">
  <video src="videos/파일명.mp4" controls controlsList="nodownload" playsinline
         onerror="this.closest('.video-wrap').classList.replace('has-video','no-video')"></video>
</div>
```

1. 영상 파일을 `videos/` 폴더에 넣기
2. `no-video` → `has-video` 로 클래스 변경
3. `src` 경로를 실제 파일명으로 수정
4. 저장 후 GitHub에 push
